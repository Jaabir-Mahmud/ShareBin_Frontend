ShareBin Snippet Feature

ShareBin now supports creating and sharing code snippets with unique URLs. Anyone can access and add code using a unique ID at the end of the URL.

How to Create a Snippet

  1. Visit ShareBin and write your code in the editor
  2. Click the Share button (paper airplane icon) in the right toolbar
  3. The snippet will be saved and a unique URL will be copied to your clipboard
  4. Share this URL with anyone

How to Access a Snippet

  1. Navigate to a snippet URL like: https://sharebin-lake.vercel.app/#/s/{unique-id}
  2. The code snippet will be loaded automatically in the editor
  3. You can edit, download, or share the snippet

Example Snippet

Try accessing this example snippet:

https://sharebin-lake.vercel.app/#/s/tuo5z01m

This snippet contains a simple JavaScript example.

Technical Details

The feature uses:

  • Unique ID generation for each snippet
  • In-memory storage on the backend (for demo purposes)
  • REST API endpoints for saving and retrieving snippets
  • Frontend routing to handle snippet URLs

For production use, the in-memory storage should be replaced with a persistent database.
